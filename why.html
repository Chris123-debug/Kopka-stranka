Trošku o Jazykoch

HTML - Základy

HTML je jazyk na tvorbu a štruktúrovanie obsahu webovej stránky. Pomocou HTML definujeme rôzne elementy stránky ako nadpisy, texty, obrázky a odkazy.

Základný HTML kód


<!DOCTYPE html>  
<html>  
<head>  
<title>Stránka</title>  
</head>
<body>  
  <h1>Toto je nadpis</h1>  
  <p>Toto je odsek textu.</p>  
  <a href="https://example.com">Kliknite sem</a>  
</body>
</html>

  • <h1> - Tento tag definuje hlavný nadpis stránky.
  • <p> - Tento tag definuje odsek textu.
  • <a> - Tento tag slúži na vytvorenie odkazu, v atribúte href sa zadáva URL adresa, na ktorú odkaz smeruje.

Vysvetlenia HTML tagov:

  • <html>: Začiatok HTML dokumentu.
  • <head>: Sekcia hlavičky dokumentu, obsahuje informácie o stránke ako metaúdaje.
  • <title>: Názov stránky, ktorý sa zobrazuje v okne prehliadača.
  • <body>: Obsah stránky, ktorý sa zobrazuje používateľovi.
  • <h1>: Hlavný nadpis, označuje najdôležitejší text.
  • <p>: Odsek textu, ktorý je bežným blokovým prvkom.
  • <a>: Odkaz, ktorý môže obsahovať URL adresu na presmerovanie.
  • <img>: Obrázok, používa atribút `src` pre určenie zdroja obrázku.

  • <div>: Rozdeľuje obsah na bloky a umožňuje nastavenie štýlu.

CSS - Základy

CSS (Cascading Style Sheets) slúži na štýlovanie vzhľadu webových stránok. Môžeme ním upravovať farby, písma, rozloženie a ďalšie vizuálne vlastnosti stránok.

Základný CSS kód


/* Nastavenie pozadia a písma pre stránku */
body {
  background-color: #f0f0f0;  /* Svetlá farba pozadia */
  font-family: Arial, sans-serif;  /* Nastavenie písma */
}

/* Nastavenie štýlu pre nadpis */
h1 {
  color: #333333;  /* Nastavenie farby písma */
  text-align: center;  /* Zarovnanie textu na stred */
}

/* Nastavenie štýlu pre odkaz */
a {
  color: blue;  /* Farba odkazu */
  text-decoration: none;  /* Odstránenie podčiarknutia odkazu */
}

Vysvetlenie: Tento kód mení pozadie na svetlo šedé a písmo na Arial. Nadpisy sú zarovnané na stred a odkazy majú modrú farbu bez podčiarknutia.

Vysvetlenie CSS tagov a vlastností:

  • background-color: Nastavuje farbu pozadia elementu.
  • color: Nastavuje farbu textu v elemente.
  • font-family: Určuje typ písma použitého v elemente.
  • text-align: Určuje, ako bude text zarovnaný v rámci elementu (napr. na stred, vľavo alebo vpravo).
  • text-decoration: Používa sa na úpravu dekorácie textu (napr. podčiarknutie, preškrtnutie).
  • padding: Nastavuje vnútorné okraje okolo obsahu elementu.
  • border: Definuje okraj elementu, jeho farbu, hrúbku a štýl.
  • display: Určuje spôsob zobrazenia elementu (blokový, in-line, flex, grid, atď.).

JavaScript - Základy

JavaScript je jazyk na pridávanie interaktivity na webové stránky. Používa sa na dynamické zmeny na stránke bez nutnosti jej načítania znova.

Základný JavaScript kód

1. Zobrazenie upozornenia

function myFunction() {
  alert("Ahoj, svet!");  // Zobrazi okno s upozornením
}

Vysvetlenie: Tento kód zobrazuje upozornenie, ktoré obsahuje text "Ahoj, svet!" keď sa spustí funkcia myFunction().

2. Zmena textu na stránke

function changeText() {
  document.getElementById("demo").innerHTML = "Text bol zmenený!";  // Zmení obsah elementu s id="demo"
}

Vysvetlenie: Tento kód mení text v HTML elemente s id "demo" na "Text bol zmenený!" keď sa spustí funkcia changeText().

3. Zmena farby pozadia

function changeBackgroundColor() {
  document.body.style.backgroundColor = "lightblue";  // Zmení pozadie stránky na svetlomodré
}

Vysvetlenie: Tento kód zmení farbu pozadia webovej stránky na svetlomodrú, keď sa spustí funkcia changeBackgroundColor().

4. Zobrazenie aktuálneho dátumu

function showDate() {
  var currentDate = new Date();  // Získa aktuálny dátum
  document.getElementById("date").innerHTML = currentDate;  // Zobrazí aktuálny dátum na stránke
}

Vysvetlenie: Tento kód získa aktuálny dátum a zobrazí ho na stránke v elemente s id "date".
