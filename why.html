Trošku o Jazykoch

HTML

HTML je značkovací jazyk na tvorbu a štruktúrovanie webových stránok.

CSS

CSS (Cascading Style Sheets) je jazyk na štýlovanie webových stránok. Umožňuje meniť farby, písma, rozloženie a responzivitu. Spolupracuje s HTML a vylepšuje vzhľad aj používateľský zážitok.

Java Script

JavaScript je programovací jazyk pre tvorbu dynamických webových stránok. Umožňuje interaktivitu, animácie, manipuláciu s HTML a CSS. Spolu s HTML a CSS tvorí základ moderného webového vývoja.

Programovanie Stránky

Programovanie webovej stránky zahŕňa HTML na štruktúru, CSS na dizajn a JavaScript na interaktivitu. Backend spracúva dáta, frontend zabezpečuje vzhľad. Moderné frameworky uľahčujú vývoj a optimalizáciu.
